## 2. ペイン分析（頻度順マトリクス）

### Tier 1: 最高頻度ペイン（6名以上が言及）

#### P1「SNS集客の限界・疲弊」（7/10名）
- ばんみき: 「1年間の虚無状態でビジネスの楽しさを完全に見失っていた」
- 白井: 「アメブロ+オーガニック集客のみで年収4,000万達成も限界を実感」
- リオナ: 「今日個別相談来るかな来ないかなの日常的不安」
- みゆ: 「売れている人を忙しくさせてしまうやり方を教えていることへの罪悪感」
- あみ: 「業界の大御所がセミナー集客一辺倒を推す中、違う方法を探していた」
- 今田: 「売れば売るほど忙しくなり受注を止めるしかない」
- ようこ: 複数のマーケターのセミナーに参加済みだが「SNSフォロワー増加中心」の内容に違和感

**→ LP訴求への落とし込み**: 「SNSを毎日投稿し続ける日々」は最も共感を得やすいペインフック

#### P2「年収の壁・天井感」（6/10名）
- ともか: 「年収2,000万円の壁を6年間破れず、やれること全部やり尽くした」
- 今田: 「年商3,000万円で成長停止、月50件のコンサルで休みなし」
- 俊介: 「紹介のみで年収2,000〜3,000万を維持していたが上が見えない」
- 白井: 「500〜600万の月収が天井」
- リオナ: 「最高月商600万円が限界」
- みゆ: 「年収4,000万は達成しているが稼働時間が比例」

**→ LP訴求への落とし込み**: 具体的な年収帯（2,000〜4,000万）を明示することでターゲットの自己認識を促す

### Tier 2: 高頻度ペイン（4〜5名が言及）

#### P3「売上と稼働時間が比例する構造」（5/10名）
- 白井/リオナ/みゆ/今田/ともか: 稼働を減らすと売上が下がる恐怖

**→ LP訴求**: 「頑張るほど忙しくなるのに天井が変わらない」

#### P4「来月の売上が読めない」（4/10名）
- 俊介: 紹介途絶への恐怖
- リオナ: 「今日来るかな来ないかな」の不安
- ばんみき: 売上の正確な数字すら把握していない
- みゆ: セミナー依存で収入が不安定

**→ LP訴求**: 「来月の売上を計算できますか？」

### Tier 3: 中頻度ペイン（3名が言及）

#### P5「広告の過去失敗・恐怖」
- 白井: 100万以上失敗のトラウマ
- リオナ: 「かけても来なかったらどうしよう」
- あみ: CPA高騰で回収不安

#### P6「他社コンサルで解決しなかった」
- ともか: 「人数を増やしなさい」としか言われない
- あみ: 過去に複数回騙された
- ようこ: 「SNSフォロワー増やしましょう」と言い出す人への警戒

## 6. LP訴求への統合設計

### ベストクライアントの「最大公約数」一言定義
> **「年収2,000〜4,000万を実力で稼いでいるのに、自分が動かないと売上が止まる構造に疲弊している専門家」**

### Pattern 1（教育型）で刺すべき訴求順序
1. **感情フック**: SNS毎日投稿の疲弊（P1） → 最も共感される
2. **数字フック**: 年収の壁（P2） → 「2,000〜4,000万で停滞」と明示
3. **信念転換**: 実力不足→構造の欠如
4. **アイデンティティ**: 8桁ファネルホルダー®️（V6が最も売れた理由）
5. **証明**: 売上増＋人生の変化（B1+B5）
6. **感動の余韻**: 「人生が動き出した」ストーリー

### Pattern 2（ストレート型）で刺すべき訴求順序
1. **具体性フック**: 7つの初期設計パーツ → 「全部やってくれるのか」の驚き
2. **構造的問題提起**: バラバラ外注/テンプレ/ツール先行の3つの間違い
3. **ROIフレーム**: 月々33,000円 vs 広告代理店月50〜200万
4. **証明**: 設計→運用→数字の因果関係（三浦: 3-4日で初回面談）
5. **安心**: AI メンター＋サクセスチーム＋コミュニティ

### 顧客事例の使い分け

#### Pattern 1向き（人生の変化ストーリー重視）
| 事例 | フック |
|------|--------|
| **ともか** | こだわりを捨てずに年収3倍。6年間の壁を突破 |
| **俊介** | 口座残高3,000円→パリコレ出演。3ヶ月連続8桁 |
| **みゆ** | セミナーゼロで8,900万。フリースクール開校 |
| **白井** | 広告100万失敗のトラウマ克服→結婚式2回 |

#### Pattern 2向き（設計→成果の因果関係重視）
| 事例 | フック |
|------|--------|
| **三浦** | 広告開始3-4日で初回面談。ROAS 1,000% |
| **リオナ** | 同じLP1年間変更なしでROAS 500-700% |
| **澤** | フレームワークをチームに渡しただけで月商2,000-3,000万 |
| **今田** | 初回ウェビナーだけで1,200万成約 |

## 7. 広告フックへの接続（Meta広告→LP一貫性）

### Pattern 1 広告フック案

- 「SNSを毎日投稿しなくても月商8桁。その仕組みの名前は—」

### Pattern 2 広告フック案
- 「ファネル構築、全部やります。月々33,000円で」
- 「LPもウェビナーも広告も。ファネルの7パーツを丸ごと初期設計」
- 「350社のファネルを設計してきた専門家が、あなたのビジネスを丸ごと設計する」
